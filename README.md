 # Daily Expense and Savings Tracker

This is a simple Python program designed to help you track your daily expenses and savings. It allows you to input your daily spending and monitor if you stay within your budget. The program gives you a summary of your expenses and remaining savings for the day.

## Features

- Enter your daily budget.
- Add multiple expenses categorized by type (like food, travel, books, etc.).
- View your total spending and see if you exceeded your budget.
- Get a summary of expenses and remaining savings at the end.

## How to Use

1. Run the Python script.
2. Enter your daily budget amount.
3. Input your expenses one by one by providing a category and amount.
4. Type `done` when finished entering expenses.
5. Review the expense summary and your savings.

## Why This Project?

This project helps beginner programmers practice basic Python concepts like loops, lists, input/output, and conditionals while solving a real-life problem — managing personal finances effectively. It’s simple, compact, and easy to customize for your own needs.

## Requirements

- Python 3.x
