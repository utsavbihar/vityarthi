# Personalized Daily Expense and Savings Tracker

## Project Overview
This is a simple Python program to help users track their daily expenses and savings. The project allows users to input various expense categories and the amount spent. It compares the total spending against a user-defined daily budget to alert when the budget is exceeded. The program also provides a summary of expenses and remaining savings at the end.

## Features
- User sets a daily budget.
- Input multiple expense categories and amounts.
- Real-time update of total spending after every expense entry.
- Alerts when spending exceeds the daily budget.
- Displays a summary report showing each expense and total savings.

## Technologies Used
- Python Basics: variables, data types, input/output
- Data structures: lists and tuples
- Control flow statements: loops and conditional checks
- Simple arithmetic operations for budget calculation
